Excercice 1: Manipuler les variables

Vous organisez une sortie scolaire pour une classe de 30 élèves.
Vous devez calculer le nombre de places nécessaires dans le bus.

Vous faites les opérations suivantes :

  • ajouter 3 parents accompagnateurs
  • retirer les 2 élèves absents
  • compter 1 place pour la maîtresse
  1. Créer une varible totalPlaces qui stocke la valeur 30 et dans laquelle vous réaliserez les opérations suivantes jusqu'à obtenir le nombre total d'élèves.
  2. Vérifier le résultat avec le console.log()
  3. Créer une variable affichageTotalPlaces en utilisant le résultat totalPlaces
La variable affichageTotalPlaces devra contenir la chaine de caractère "Il faudra totalPlaces dans le bus"
