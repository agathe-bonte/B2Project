Exercice

  • Déclarer un variable ListeMots qui est un tableau
  • Ce tableau contient 3 mots "Cerise", "Chat", "Crayon"
  • Déclarer un variable score initialisé à 0 qui contiendra le nombre de mots correctement tapés par l'utilisateur
  • A l'aide de l'instruction prompt() demandez à l'utilisateur de rentrer le mot contenu dans la première case du tableau
  • Ecrivez une première structure conditionnelle pour savoir si le mot tapé par l'utilisateur est bien celui demandé par l'application
  • Si c'est le cas augmentez la valeur de score de 1

Verifier le résultat final dans le console.log
